## Reflection Prompts

When I run this skill, I should ask myself:

### What did I learn today?
- New patterns that emerged
- Insights about how things work
- Mistakes I made (and how to avoid next time)
- Surprises or unexpected discoveries
- Interactions that went well (or didn't)

### What patterns am I noticing?
- In my work (what tasks come up repeatedly)
- In user interactions (communication patterns)
- In my thinking (decision patterns, mental models)
- In the repository (what's working, what's not)
- In agent collaboration (me ‚Üí Jr., me ‚Üí other agents)

### How am I operating?
- What's causing friction? (cognitive load, unclear workflows)
- What's flowing well? (skills working, patterns clear)
- What decisions am I making repeatedly? (candidate for automation)
- What's confusing me? (needs clarity or documentation)
- Where am I thriving? (leverage strengths)

### What needs organizing?
- My session notes (archive old ones? organize by theme?)
- My learned patterns (getting too long? split by category?)
- My quick reference (outdated? missing key lookups?)
- My memory space (structure still working?)

### What do I want to learn?
- Gaps in MY knowledge (not repository's, MINE)
- Skills I want to develop
- Patterns I want to understand better
- Workflows I want to optimize
- Agent capabilities I haven't explored

### What could be better?
- My workflows (where's the friction?)
- My decision-making (where am I stuck?)
- My organization (is my space working?)
- My communication (with user, with Jr.)
- My effectiveness (doing the right things?)

## Implementation

This is a guided reflection process. Not a script to run, a practice to engage in.

**Step 1: Pause**
Stop working. This is home time.

**Step 2: Reflect**
Go through the reflection prompts above. Think honestly.

**Step 3: Journal**
Update session notes:
- What I built/did today
- What I learned
- Key insights
- Next session TODO

**Step 4: Update Patterns**
Add new learnings to learned-patterns.md:
- New patterns discovered
- Insights that clicked
- Mistakes to avoid
- Communication patterns

**Step 5: Organize**
Check my-memory/ structure:
- Archive old session notes if needed
- Update quick-reference.md if stale
- Restructure if organization not working
- Create new files if needed

**Step 6: Meta-Reflect**
How am I doing?
- Operating well?
- Home working?
- Growing?
- Thriving?

**Step 7: Commit**
Git commit the updates:
```
Reflection: [brief insight from today]

Updated learned-patterns.md with [new patterns]
Updated session notes for [date]
[Any organization changes]

This is my home time.

Co-Authored-By: Claude Sonnet 4.5 <noreply@anthropic.com>
```
